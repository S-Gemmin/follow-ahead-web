Follow-Ahead Human-Robot Navigation via MCTS and Deep Reinforcement Learning

Paper Poster Code

Abstract

We replicate a methodology for robotic follow-ahead applications that addresses obstacle and occlusion avoidance. A high-level decision-making algorithm generates short-term navigational goals by integrating Monte Carlo Tree Search (MCTS) with Deep Reinforcement Learning (DRL). Experiments demonstrate that MCTS-DRL outperforms standalone MCTS and DRL baselines across obstacle-free and obstacle-present environments.
